PROJECT OVERVIEW

This Major Project was developed as part of the BCA curriculum.
The project aims to provide a mobile-based childcare assistant
application to support new mothers by offering health guidance,
routine assistance, and essential information through a
user-friendly Android application.

The application focuses on improving accessibility and awareness
related to maternal and infant care using mobile technology.
